# savage-ai-im
- An exploration into LangChain/LangGraph and LLM powered bot software using [savage-aim](https://savageaim.com) as a basis.
  - For more info, see `~/projects/savageaim.md`
- It is a very simple Discord Bot that uses LangChain on top of VertexAI to take plain-english requests from people and parses data from the SavageAim API to answer them.
- As a brief foray into the world of LangChain and tool-calling, it is extremely basic, but it's public nonetheless!

- [GitHub](https://github.com/freyamade/SavageAI-IM)
